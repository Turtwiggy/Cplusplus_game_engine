### More Info

This project uses 

- SDL2 - Window management
- Opengl/Glew - OpenGL extention wrangler
- ImGui 
- ImGuizmo

Maybe in future

- Entt - entity component system
- GGPO
- Bullet3 - physics
- GameNetworkingSockets
